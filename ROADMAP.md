# Roadmap

## v0.1 - Project Foundation

- Create project structure.
- Add README, roadmap, changelog, project checklist, and version file.
- Prepare folders for databases, Excel files, source code, and documents.

## v0.2 - First Equipment Database

- Add Deye LV inverter database.
- Add Jinko panel database.
- Add Longi panel database.
- Define common equipment fields.

## v0.3 - First Excel Calculator

- Create first Excel workbook.
- Add basic system input screen.
- Add first compatibility checks.
- Add embedded starter equipment database sheets.

## v0.4 - MPPT and String Sizing

- Add dropdown equipment selectors.
- Calculate recommended min/max panels per string.
- Add clearer result formatting.
- Add database filters and frozen headers.

## v0.5 - Mounting Module

- Add mounting module for roof type and mounting quantity calculation.
- Add hooks, rails, clamps, bolts, grounding, and cable clip estimates.
- Add mounting rules database.

## v0.6 - Compatibility Matrix

- Add selected-system compatibility matrix.
- Add PASS, FAIL, and VERIFY status logic.
- Add datasheet completeness warning.
- Add inverter-panel and inverter-battery compatibility summary.

## v0.7 - Summary Sheet

- Add quick summary sheet.
- Add PASS/FAIL/VERIFY conditional formatting.
- Add battery and mounting summary values.

## v0.8 - Equipment Output

- Add basic equipment quantity output.
- Add selected inverter, panels, battery, and mounting quantities.
- Add equipment item count to summary.

## v0.9 - Cables and Protection

- Add cable and protection device outputs.
- Add starter DC/AC cable quantities.
- Add DC/AC protection quantities.

## v0.10 - Cable and Protection Sizing

- Add cable section calculations.
- Add breaker and fuse rating calculations.
- Add starter voltage-drop estimates.

## v0.11 - Advanced Protection

- Add SPD type selection.
- Add earthing system details.
- Add installation derating factors.

## v0.12 - Bulk Matrix

- Add bulk compatibility table for common inverter, panel, and battery combinations.
- Add batch PASS / FAIL / VERIFY outputs.

## v0.13 - Yield and Economics

- Add regional solar yield assumptions.
- Add tariffs and green tariff scenarios.
- Estimate annual generation.
- Estimate savings and payback.

## v0.14 - Customer Option Sizing

- Add monthly consumption input.
- Add Premium, Standard, and Economy system options.
- Add day-night tariff comparison.
- Add green/export revenue.
- Add 20-year degradation estimate.

## v0.15 - Monthly Yield and Tariff Verification

- Add monthly generation profile.
- Add supplier/date tariff verification workflow.
- Add separate self-consumption and export revenue lines.

## v0.16 - Reports

- Generate PDF summary.
- Generate commercial proposal.
- Add project notes and assumptions.

## Later

- Add more inverter brands.
- Add more panel brands.
- Add roof and mounting calculations.
- Add cable and protection calculations.
- Add web or desktop interface if Excel becomes limiting.
