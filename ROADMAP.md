# Roadmap

## v0.1 - Project Foundation

- Create project structure.
- Add README, roadmap, changelog, project checklist, and version file.
- Prepare folders for databases, Excel files, source code, and documents.

## v0.2 - First Equipment Database

- Add Deye LV inverter database.
- Add Jinko panel database.
- Add Longi panel database.
- Define common equipment fields.

## v0.3 - First Excel Calculator

- Create first Excel workbook.
- Add basic system input screen.
- Add first compatibility checks.
- Add embedded starter equipment database sheets.

## v0.4 - MPPT and String Sizing

- Add dropdown equipment selectors.
- Calculate recommended min/max panels per string.
- Add clearer result formatting.
- Add database filters and frozen headers.

## v0.5 - Mounting Module

- Add mounting module for roof type and mounting quantity calculation.
- Add hooks, rails, clamps, bolts, grounding, and cable clip estimates.
- Add mounting rules database.

## v0.6 - Compatibility Matrix

- Add compatibility matrix for common Deye, Jinko, and LONGi combinations.
- Add PASS/FAIL conditional formatting.
- Add quick summary sheet.
- Add basic equipment quantity output.

## v0.7 - Yield and Economics

- Add regional solar yield assumptions.
- Add tariffs and green tariff scenarios.
- Estimate annual generation.
- Estimate savings and payback.

## v0.8 - Reports

- Generate PDF summary.
- Generate commercial proposal.
- Add project notes and assumptions.

## Later

- Add more inverter brands.
- Add more panel brands.
- Add roof and mounting calculations.
- Add cable and protection calculations.
- Add web or desktop interface if Excel becomes limiting.
