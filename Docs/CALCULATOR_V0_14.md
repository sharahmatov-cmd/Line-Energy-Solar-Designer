# Excel Calculator v0.14

File:

`Excel/Line-Energy-Solar-Calculator-v0.14.xlsx`

## Added in v0.14

- Monthly consumption input.
- Target coverage input, default 70%.
- Roof type / ground mount selector.
- Day-night tariff comparison inputs.
- `SystemOptions` sheet with Premium, Standard, and Economy options.
- Automatic option sizing to cover the target share of annual consumption.
- Daily, monthly, and annual generation per option.
- Single-rate savings, day-night savings, and green/export revenue per option.
- Estimated option cost and simple payback.
- `Lifetime` sheet with 20-year generation degradation.
- `OptionTiers` sheet with cost, service life, and degradation assumptions.

## Current Limitation

The options are preliminary commercial sizing outputs. Cost per kWp, tariff values, self-consumption share, installation type, and equipment tier assumptions must be verified for each project before customer proposal use.
