# Excel Calculator v0.7

File:

`Excel/Line-Energy-Solar-Calculator-v0.7.xlsx`

## Added in v0.7

- `Summary` sheet as the first workbook screen.
- Compact selected-system overview.
- Overall compatibility status.
- PV status, battery status, and datasheet completeness status.
- Battery energy summary.
- Mounting quantity summary.
- Conditional formatting for `PASS`, `FAIL`, and `VERIFY` status cells.

## Status Colors

- `PASS` - green.
- `FAIL` - red.
- `VERIFY` - yellow.

## Purpose

This version makes the workbook easier to read as a working calculator. The detailed sheets remain available, but the first screen now shows the main result and key quantities.
