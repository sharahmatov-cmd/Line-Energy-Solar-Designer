# Mounting Module

The mounting module calculates starter quantities for solar mounting components by roof type.

## Supported Roof Types

- Metal tile
- Standing seam
- Trapezoidal sheet
- Flat roof

## Components

- Roof hooks or seam clamps
- Mini-rails
- Rails
- Rail connectors
- End clamps
- Middle clamps
- Bolt sets
- Grounding clips
- Cable clips

## Calculation Inputs

- Roof type
- Total panels
- Panels per row
- Number of rows
- Rail stock length
- Rail length per panel
- Safety reserve

## Important Limitation

The current rules are starter assumptions. Final mounting quantities must be checked against the selected mounting system, roof geometry, wind/snow zone, rail span tables, and manufacturer installation instructions.
